# v0.1.0

Initial public experimental release of `design-architecture`.

## What is included

- portable skill payload under `skill/`
- conservative Mermaid validation for the default flowchart subset
- optional explicit C4 mode for selected diagram types
- derived `.excalidraw` generation with experimental status
- architecture drift detection with simple, conservative heuristics
- fixtures, smoke tests, and automated tests for the current v1 behavior
- Codex installation documentation and public repository onboarding

## Current expectations

- Mermaid remains the canonical source of truth
- default mode is standard Mermaid flowchart
- C4 support is explicit, optional, and narrower than the default mode
- `.excalidraw` output is still experimental and not yet claimed as officially import-verified
- drift detection is intentionally heuristic and conservative

## Install on Codex

```bash
$skill-installer install https://github.com/go2ai/architecture-diagram-skill/tree/main/skill --name design-architecture
```

Restart Codex after installation.
